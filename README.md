# 神州租车全国搜车工具

一个基于神州租车（zuche.com）H5 网关接口的桌面小工具，用于**一键扫描全国所有开放城市**的车辆库存与价格，并支持按车型筛选、表格查看、全国地图可视化和 CSV 导出。

> 本工具通过复用你本人登录后的浏览器 Cookie 调用神州租车 H5 站点的公开接口，仅用于个人查价比价，不涉及任何账号密码或支付信息。请遵守神州租车官网的服务条款，合理控制请求频率，仅供学习交流使用。

## 功能特性

- 🔍 **全国扫描**：输入 Cookie、起租日期与租期天数，自动遍历全部开放城市，抓取各网点车型与价格
- 📊 **结果查询**：按车型多选筛选，表格展示城市 / 网点 / 地址 / 配置 / 日租 / 套餐价 / 预估总价，并自动排序取最低价
- 🗺️ **地图可视化**：基于 ECharts 生成中国地图热力散点图，直观查看各城市价格分布
- 📥 **CSV 导出**：导出筛选后的结果表格，方便进一步分析
- 💾 **本地缓存**：自动缓存 Cookie 与上次扫描结果，下次打开自动加载
- 🚦 **限速与重试**：内置请求频率限制与失败自动重试，降低触发风控概率

## 环境要求

- Windows 系统
- Python 3.8+（需自带 `tkinter`，官方安装包默认包含）
- 依赖库：`requests`（见 `requirements.txt`）

## 快速开始

### 方式一：双击启动脚本（推荐）

直接双击 [`start.bat`](start.bat)，脚本会自动检查 Python 环境、安装缺失依赖并启动程序。

### 方式二：命令行启动

```bash
pip install -r requirements.txt
python main.py
```

## 使用说明

1. 打开神州租车 H5 站点（`https://m.zuche.com`）并登录，用浏览器开发者工具复制请求头中的完整 `Cookie`（需包含 `intranet-sessionid`）
2. 将 Cookie 粘贴到程序「全国扫描」页的输入框中
3. 按需设置代理、起始日期、租期天数
4. 点击「🚀 全国扫描」，等待扫描完成（可随时点击「⏹ 停止」中断）
5. 切换到「📊 结果查询」页，在左侧车型列表中多选车型（按住 Ctrl 多选），点击「📊 查看表格」或「🗺️ 显示地图」查看结果，也可点击「📥 导出CSV」导出数据

## 项目结构

```
.
├── main.py               # 主程序（Tkinter GUI + 扫描 / 查询 / 地图 / 导出）
├── start.bat              # 一键启动脚本
├── requirements.txt       # Python 依赖
├── city_fields.json       # 城市接口字段参考样例
├── check_city_api.py      # 调试：校验城市列表接口返回字段
├── check_coords.py        # 调试：查看扫描结果中的坐标数据
├── discover_api.py        # 调试：探测网关接口
├── find_api.py             # 调试：从前端 JS 中查找 API 端点
├── find_cities.py          # 调试：获取完整开放城市列表
├── read_result.py          # 调试：读取分析结果文件
└── test_api*.py / test_direct.py  # 调试：接口调用格式验证脚本
```

`check_*` / `discover_*` / `find_*` / `test_*` 系列脚本是逆向分析接口时使用的调试脚本，非运行主程序所必需，仅作为参考保留。
